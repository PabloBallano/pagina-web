Estructura del sitio web

El sitio web está organizado en las siguientes secciones:

  • Grado: Descripción del grado y de las asignaturas.
  • Fundamentos de Ingeniería Informática: Información detallada sobre esta asignatura.
  • Tema: Desarrollo de un tema de interés relacionado con la informática.
  • Red: Enlaces a las páginas web de compañeros.
